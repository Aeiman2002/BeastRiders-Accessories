## Project Overview

Beast Riders Accessories is an e-commerce web application designed for riders to explore and purchase accessories for heavy bikes.

## The site includes the following:

- Home page with product showcase and call-to-action for bikes/accessories.  
- Product catalog grouped by major motorcycle brands (Kawasaki, Yamaha, Honda, etc.).  
- Login and cart interfaces.  
- Contact form with support information.  
- Brand and category filtering for quick access.
